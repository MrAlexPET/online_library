Технологический стек проекта:
1. Django - Это фреймворк для создания веб-приложений на языке Python.
2. Python: Язык программирования, на котором написано веб-приложение.
3. MySQL: База данных MySQL используется для хранения данных приложения.
4. HTML/CSS: Для создания пользовательского интерфейса веб-приложения используются HTML (язык разметки) и CSS (язык стилей).
6. Git/GitHub: Система контроля версий.
